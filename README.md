# beer-shop
An online shop application, the complete microservices demo for kratos.

## project structure
```
.
├── api  // API&Error proto&generated code
│   ├── foo
│   │   └── service
│   ├── bar
│   │   └── interface
├── app  // kratos microservices projects
│   ├── foo
│   │   └── service
│   ├── bar
│   │   └── interface
├── pkg  // common used packages
├── web  // web ui projects
│   ├── admin  // for management
│   └── shop   // for customer
├── docs
└── deploy
    ├── build
    ├── docker-compose
    └── kubernetes
```
